# Squirrel Video Frame Interpolation
- BY Squirrel Development Team
- 基于RIFE算法的补帧软件

## 特点

- 效果更好，显存占用更小，是DAIN速度的十五倍，包含抽帧处理，可去除动漫卡顿感

- 一键队列运行，支持多卡，持续更新

## 软件环境需求
（1） .NET Framework 4.5.2及以上版本

（2） Microsoft Visual C++ Redistributable 2019

（3） 最新的NVIDIA显卡驱动

（4） 足够的硬盘空间

（5） 4G以上内存大小

如果无NVIDIA显卡，可选用CPU补帧
