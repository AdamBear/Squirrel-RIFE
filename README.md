# Squirrel-RIFE Video Frame Interpolation
- BY Squirrel Development Team
- 基于RIFE算法的中文补帧软件

# 项目分为Squirrel-RIFE(前SVFI版), RIFE GUI 两个版本

## 特点

- 效果更好，显存占用更小，是DAIN速度的十五倍

- 包含抽帧处理，可去除动漫卡顿感 (SVFI版本)
