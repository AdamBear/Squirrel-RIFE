# Squirrel-RIFE Video Frame Interpolation
 - BY Squirrel Development Team
 - 基于RIFE算法的中文补帧软件

----

## ✨ 特点

- 效果更好，显存占用极小，是DAIN速度的二十倍

- 包含抽帧处理，可去除动漫卡顿感

## 🖥 系统要求
- Windows 7 及以上操作系统

- NVIDIA 显卡以及 GeForce Experience 所安装的驱动

- Microsoft Visual C++ 2015-2019 Redistributable

- 至少 2GB 以上显存以及 4GB 左右的空余内存
